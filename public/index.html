Responsiivinen suunnittelu

 Mitä on responsiivinen suunnittelu?

Responsiivinen suunnittelu on suunnittelua, joka tavoite on mahdollistaa nettisivun tai applikaation käyttö niin kaikille laitteille kuin kaikille näyttökoille. Vaikka pöytätietokoneiden ja kannettavien tietokoneiden näytön resoluutiot ovatkin kasvaneet huomattavasti, on samaan aikaan mobiililaitteiden käyttö internetin selaamiseen yleistynyt.

 Perinteisesti mobiililaitteille on luotu erillinen versio tai natiivisovellus palvelusta, mutta  jatkuvasti kehittyvä web-teknologia on mahdollistanut yhden version tarjoamisen kaikille laitteille.  Tämä nopeuttaa uusien palvelujen kehittämistä ja helpottaa myös käyttäjän elämää.

Responsiivista suunnittelua voidaan toteuttaa muun muassa näytön kokoon mukautuvien elementtikehikoiden avulla, kuvilla  joiden koko on relatiivisesti määritetty niitä ylempien elementin koon mukaan ja CSS media queryillä jotka mahdollistavat CSS - määritysten asettamisen näyttökoon mukaan.

 CSS Media query

CSS Media queryjen avulla voidaan asettaa ulkoasulle useita halkeamispisteitä ns. "breakpointteja", joihin voidaan asettaa määrityksiä tietynlaiselle näyttötyypille tai -koolle.
Breakpointit määritellään muiden tyylimääritysten sekaan hieman erikoisfonttien määrittelyä vastaavalla syntaksilla

@media all and (max-width: 960px) {
  /* 
    Määritetään .column - luokan omaavat elementit puoleen näytön koosta
    kun näytön koko on maksimissaan 960 pikseliä leveä
  */
  .column {
    width: 480px;
  }
}

Queryjen määrittelyssä asetetaan ensin minkälaiselle laitteelle (medialle) asetus on tarkoitettu. Tässä tapauksessa määritys koskee kaikkia laitteita (all). Muita usein käytettyjä määrityksiä:

  • handheld - Puhelimet, tabletit
  • screen - Pöytätietokoneet ja kannettavat tietokoneet
  • tv - Televisiot

Kaikki määrityksen ehdot yhdistetään loogisilla operaattoreilla and, pilkku (, joka vastaa käytännössä OR operaattoria) , not, only

Yleisimpiä näytön ominaisuuksien ehtoja:

  • max-width - Ikkunan maksimileveys
  • max-height - Ikkunan maksimikorkeus
  • min-width - Ikkunan minimileveys
  • min-height - Ikkunan minimikorkeus
  • orientation - Laitteen orientaatio (landscape|portrait)
  • aspect-ratio - Ikkunan kuvasuhde
@media screen and (max-width: 995px) , screen and (max-height: 700px) {
  /* Esimerkki määrityksien ketjuttamisesta */
  #cat {
    display: none;
  }
}

Lisää tietoa media queryistä löytyy ominaisuuden virallisesta määrittelystä tai Mozillan kehittäjäsivulta

Internet Explorer tukee media queryjä vasta versiosta 9 eteenpäin
